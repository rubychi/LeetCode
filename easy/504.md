# Description

Given an integer, return its base 7 string representation.

**Example 1:**
<pre>
<b>Input:</b> 100
<b>Output:</b> "202"
</pre>

**Example 2:**
<pre>
<b>Input:</b> -7
<b>Output:</b> "-10"
</pre>

**Note:** The input will be in range of [-1e7, 1e7].

## Solution
```javascript
/**
 * @param {number} num
 * @return {string}
 */
var convertToBase7 = function(num) {
    if (num === 0) {
        return "0";
    }
    let negFlag = false;
    if (num < 0) {
        num = -num;
        negFlag = true;
    }
    let result = "";
    while (num) {
        result += (num % 7);
        num = Math.floor(num / 7);
    }
    result = result.split("").reverse().join("");
    return negFlag ? "-" + result : result;
};
```
